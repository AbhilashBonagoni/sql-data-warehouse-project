# Data Warehouse and Analytics Project 🚀

Welcome to the **Data Warehouse and Analytics Project** repository! This project demonstrates a comprehensive data warehousing and analytics solution, from building a data warehouse to generating actionable insights. Designed as a portfolio project, it highlights industry best practices in data engineering and analytics.
---

## 📋 Project Requirements & Specifications

### 🛠️ 1. Building the Data Warehouse (Data Engineering)
*   **Objective:** Develop a modern data warehouse using SQL Server to consolidate sales data, enabling analytical reporting and informed decision-making.
*   **Data Sources:** Multi-source ingestion combining two distinct systems (**ERP** and **CRM**) provided as flat CSV files.
*   **Data Quality:** Intensive cleansing pipelines programmed to profile, validate, and resolve deep data quality issues and discrepancies prior to downstream analysis.
*   **Integration:** Merge and transform separate system data into a single, user-friendly relational data model (Star Schema) optimized for analytical and aggregate querying.
*   **Scope:** Focus strictly on processing the latest operational dataset; historical data tracking (SCD type tables) is outside the current scope.
*   **Documentation:** Comprehensive documentation of data mapping architectures, business logic rules, and schema layouts to support both technical teams and non-technical business stakeholders.

### 📊 2. BI: Analytics & Reporting (Data Analytics)
*   **Objective:** Develop advanced SQL-based analytics and self-service reporting layers to deliver detailed operational insights into **Customer Behavior**, business performance, and strategic sales trends.

---

## 🛠️ Technical Skillset & Toolkit Applied

*   **Database Programming & Engineering:** Advanced SQL (T-SQL / SQL Server), complex multi-table joins, subqueries, view optimization, data extraction, transformation, and structural loading layers.
*   **Data Quality & Governance:** Comprehensive QA/QC frameworks, data profiling, lineage validation, consistency auditing, and proactive resolution of data abnormalities.
*   **Business Intelligence (BI):** Microsoft Power BI (Dataflows, semantic modeling, performant DAX calculations, and Power Query/M-code data ingestion rules).
*   **Advanced Analytics & Programming:** Scripted automation using Python (Pandas & NumPy for data cleaning and exploration) and Base SAS programming.
*   **Methodologies:** Agile workflow execution, business requirement analysis, process mapping, and Lean Six Sigma continuous improvement strategies.
